# EDA de ventas Olímpica

Este Jupyter Book presenta un análisis exploratorio de datos robusto sobre la base de ventas entregada para el proyecto de analítica.

El objetivo principal es entender la estructura de los datos, evaluar su calidad, analizar patrones temporales, estudiar el comportamiento por punto de venta, categorías, productos, promociones, tickets y casos especiales.

## Contenido del análisis

El libro está organizado en los siguientes capítulos:

1. Configuración del proyecto
2. Carga y revisión inicial de la base
3. Limpieza y calidad de datos
4. Análisis temporal
5. Análisis por punto de venta
6. Análisis de categorías y productos
7. Análisis de promociones y descuentos
8. Análisis de tickets y facturas
9. Outliers y casos especiales
10. Conclusiones del EDA
